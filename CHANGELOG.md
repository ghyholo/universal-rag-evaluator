# Changelog

## 0.2.1 - 2026-06-30

- Compare heterogeneous benchmarks with metric-specific paired samples.
- Report how many shared queries were excluded for each metric.
- Add `python -m rageval_lab` entry point and installation-free `make test`.
- Add regression tests for partially labeled comparisons.

## 0.2.0

- Added Base/Oracle/Retrieved diagnostics.
- Added slice-level aggregation and Markdown reports.
- Added temporal, outdated, distractor, grounding, and stage-latency metrics.
- Added multi-metric paired comparison with Holm correction.
- Added model-judge calibration with Cohen's kappa.
- Expanded validation, tests, documentation, examples, and Codex Skill.

## 0.1.0

- Initial framework-agnostic JSONL evaluator.
