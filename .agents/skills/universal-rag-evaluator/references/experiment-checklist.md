# Experiment checklist

- [ ] Frozen benchmark and corpus hashes
- [ ] Unique query IDs and document IDs
- [ ] Gold evidence and atomic required claims
- [ ] Unanswerable and adversarial queries
- [ ] Retrieval baselines
- [ ] Base/Oracle/Retrieved conditions
- [ ] Fixed context budget
- [ ] Per-query result files
- [ ] Slice-level analysis
- [ ] Paired confidence intervals and significance tests
- [ ] Latency, token, and cost report
- [ ] Judge calibration subset
- [ ] Reproducibility manifest
